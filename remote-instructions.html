Remote desktop software allows us to temporarily take control of your PC in order to diagnose and repair any problems without needing to arrive on-site!


This can save you time and money by avoiding unnecessary transport and time costs.


The only requirements are that you have a Windows PC, Google Chrome, and an internet connection.

Step 1.


Go to the Chrome Web Store or click the link below and install Google's remote desktop app.


Install from the Chrome Web Store here

Step 2.


Open the app and select the green "Share" button as shown in the image below.

Step 3.


Once the app gives you an access code, send the code to us via email or phone call.

NOTE: Your access code only works while you are actively waiting for the remote connection, and a new one is created with each session. This means you don't have to worry about the code being used by someone else to gain access to your PC.
